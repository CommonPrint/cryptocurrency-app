# Описание
Веб-приложение для отображения списка актуальных криптовалют, где пользователь может получить детальную информацию о каждом криптотокене. 

# Технологии
- ReactJS
- TypeScript
- Redux Router v6
- Redux toolkit
- Node.js
- CoinMarketcap API (получение данных о криптовалютах)
